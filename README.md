# AI-Based Network Anomaly Detection Using NetFlow

## Overview

This project presents a complete AI-powered Intrusion Detection System based on NetFlow traffic analysis.

The solution combines:

- NetFlow/IPFIX
- CICFlowMeter
- Wazuh SIEM
- Transformer Neural Networks
- Real-time Alerting
